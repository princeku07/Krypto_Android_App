
# Krypto
Crypto Currency maket news and Currency information app.


## Tech Stack

- Jetpack Compose
- Retrofit
- Dependency Injection (Hilt)
- Lottie animation
- Glide for image loading
**Architecture**
- Clean Architecture
- MVVM
